# Análisis de Feedbacks con IA

## Descripción

Proyecto académico desarrollado en Henry para automatizar el análisis, clasificación y ruteo de feedbacks de clientes mediante IA y herramientas No-Code / Low-Code.

El sistema procesa mensajes de clientes, detecta sentimiento, clasifica la categoría del feedback y deriva cada caso al flujo correspondiente.

## Problema

Las empresas suelen recibir feedbacks de clientes por distintos canales y analizarlos de forma manual. Esto puede generar demoras, pérdida de trazabilidad, errores de clasificación y falta de seguimiento en casos críticos.

## Solución

Se diseñó una automatización con n8n, Supabase, Airtable, Google Sheets e IA para procesar feedbacks, enriquecer datos, clasificar mensajes y registrar cada acción tomada.

## Funcionalidades principales

- Recepción de feedbacks de clientes.
- Enriquecimiento de datos desde Supabase.
- Clasificación automática con IA.
- Detección de sentimiento.
- Ruteo según categoría detectada.
- Registro de logs en Google Sheets.
- Gestión de casos técnicos, reclamos, sugerencias y clientes satisfechos.
- Alertas internas y seguimiento de casos pendientes.

## Categorías utilizadas

- Satisfacción
- Sugerencia
- Observación
- Falla técnica
- Reclamo
